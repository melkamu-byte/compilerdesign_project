Assignment_03 – Semantic Analysis
Topic: Manage scopes involving try/catch/finally
Objectives Covered:
- Symbol table and scope management
- Semantic error detection
- Attribute grammar application
- Exception-handling semantics
- Robust semantic analysis before code generation
Description:
This assignment explains how a compiler’s semantic analyzer manages
scopes for exception-handling constructs such as try, catch, and finally.
It ensures correct scoping of exception variables and validates proper
resource access.
